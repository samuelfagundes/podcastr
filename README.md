# podcastr

Plataforma para ouvir seus podcasts favoritos

## Rodando localmente

Para rodar localmente essa aplicação faça o seguinte:

### Copie o projeto

```
git clone https://github.com/samuelfagundes/podcastr/ podcastr
```

```
cd podcastr
```

### Instale as dependencias

```
yarn install
```

### Rode o server

```
yarn server
```

### Rode a aplicação

```
yarn dev
```
